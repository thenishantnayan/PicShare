# PicShare

A full-stack MERN image sharing application.

## Features
- Upload images
- Add captions
- View feed
- Cloud image storage with ImageKit
- MongoDB Atlas database

## Tech Stack
- React
- Node.js
- Express.js
- MongoDB
- ImageKit
- Multer

## Installation

### Backend
cd Backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
